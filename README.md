# 🥋 Project Ronin

> **My personal Linux workstation toolkit built from scratch.**

Project Ronin is a terminal-based dashboard and productivity toolkit written entirely in **Bash**. It was created to simplify my daily Linux workflow by bringing system monitoring, development tools, homelab utilities, and learning resources into one place.

---

## 🚀 Features

* 🩺 **Doctor** — System health and diagnostics
* 🌐 **Network** — Network information and connectivity tools
* 🤖 **AI** — Launch my local AI workspace
* 🐳 **Docker** — View and manage Docker containers
* 📝 **Ronin Notes** — Open my Obsidian knowledge vault
* ☀️ **Weather** — Dynamic weather with automatic location detection and UV index
* 📚 **Ronin Academy** — Built-in Linux learning module
* 🚀 **Ship** — One-command Git workflow for staging, committing, and pushing changes
